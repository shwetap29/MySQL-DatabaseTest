# 🗄️ MySQL Database Test — SQL Query Practice

> A hands-on SQL practice project covering essential database query patterns — from basic SELECT statements to joins, aggregations, and subqueries — built to strengthen foundational database skills.

---

## 📌 About This Project

This repository was created during my early learning phase as a **Software Engineering Apprentice at Bridgelabz (2021–2022)**. The goal was to get comfortable writing and executing real SQL queries against a MySQL database — building the kind of database fluency that I now use daily in Business Analysis and Data Analytics work.

**Honest framing:** This is a foundational learning project. The queries here are practice exercises, not a production system. I'm keeping it public because SQL is at the core of everything I do as a BA and DA — and this is where that skill started.

---

## 🎯 What's Covered

The project covers a range of SQL operations, progressing from basic to more complex:

**Basic Operations**
- `SELECT` with `WHERE`, `ORDER BY`, `LIMIT`
- Filtering with `AND`, `OR`, `NOT`, `IN`, `BETWEEN`
- Working with `NULL` values and `IS NULL` checks

**Aggregations & Grouping**
- `COUNT`, `SUM`, `AVG`, `MIN`, `MAX`
- `GROUP BY` and `HAVING` clauses
- Aggregate filtering on grouped results

**Joins**
- `INNER JOIN` — matching records across tables
- `LEFT JOIN` — including unmatched records from the left table
- `RIGHT JOIN` — including unmatched records from the right table
- Multi-table joins with aliasing

**Advanced Queries**
- Subqueries (nested SELECT statements)
- `CASE WHEN` conditional logic
- `DISTINCT` for deduplication
- String functions: `CONCAT`, `UPPER`, `LOWER`, `SUBSTRING`
- Date functions: `NOW()`, `DATEDIFF`, `DATE_FORMAT`
